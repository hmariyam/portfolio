PROJETS


Voici quelques projets sur lesquels j'ai travaillé au cours de mon parcours en tant qu'étudiante en informatique <3
Le code est disponible sur mon compte github!!




Cowboy Hero

Plonger dans une aventure désertique avec Hero le Cowboy! Ce jeu platforme contient une partie 2D et 3D. Celui-ci à été développé avec Unity 2021 en C# pour mon cours d'application jeu et simulation.

Green Pulse

Un site web transactionel pour la gestion des offres de grains céréales, légumineuses, etc. Celui-ci a été développé en Springboot pour mon cours d'application web 2.

Audio Library

Une application mobile qui permet d'ajouter, de modifier ou de supprimer des chansons. Ceci a été développé avec Android Studio en Kotlin.

Tic Tac Toe

Un jeu classique de Tic Tac Toe avec un joueur humain et un COM. Ce jeu est disponible en anglais et en espagnol. Ceci a été développé avec Android Studio en Kotlin.

Sweet&Sour Pomodoro

Un chronomètre pomodoro en thème de fruit. Ceci a été développé avec ElectronJS, HTML, CSS et JS.
